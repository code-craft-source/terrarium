Varieties and Subclasses


This project is a continuation of our terrarium project. If you haven't completed that project yet, please begin there

A subclass is an extension of another class in JS. By default, they inherit all properties and functions from their superclass, but these can be overriden to produce new behaviour and other functionality can be added. Check out the AlleyCat (subclass of Cat) and Farmer (subclass of Person) classes for examples.

Directions
  1. Let's first make some updates to our code to make subclassing easier. Create two new functions: showTrunk() (which shows the tree's trunk), and showLeaves() (which shows the tree's leaves). Call these new functions appropiately in your show() function ensure your forest functions the same way, and show your instructor.

  2. Now we can create our first subclass of our Tree. Each tree subclass should represent a variety of tree. Let's begin by creating a new EvergreenTree subclass. We'll modify our showLeaves() function to use p5's triangle() to create their unique appearance. Your EvergreenTree class should look something like this:

    // Our new subclass EvergreenTree, inherits functions and properties from its superclass Tree class EvergreenTree extends Tree { // This will overwrite Tree's showLeaves() showLeaves() { // Write new draw code to create an evergreen's unique appearance } }

  3. As you create trees in setup(), randomly select create either a Tree or an EvergreenTree. Confirm that your forest has a variety of both trees.

  4. Now let's add another variety of tree, MossyTree. This time we will overwrite Tree's' showTrunk() function to create a green trunk. Your MossyTree class should look something like this:

    // Our new subclass MossyTree, inherits functions and properties from its superclass Tree class MossyTree extends Tree { // This will overwrite Tree's showTrunk() showTrunk() { // Write new draw code to create a green, mossy trunk } }


  5. In your setup() function, add MossyTree to the list of trees that can be randomly selected.

  6. Come up with a new variety of tree and create a subclass. Be creative, but think of what properties you can change to generate a new tree. You could add color, leaves, change the the size, anything! Just be sure to allow that tree to be selected in your setup() functions.

  7. Your forest should now be full of 4 different trees that each have a unique look! Next we'll observe the passage of time on your trees.
