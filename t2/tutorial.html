Terrarium


Here is a fun starter game to get familiar with P5 and Javascript classes. Create a random forest from simple shapes .

A class is a custom object that we will define and use, as opposed to classes that come built into JS like numbers or arrays. Check out the Cat and Person classes for examples.

Directions
  1. Set up your HTML page, insert the p5.js source tags into the <head> section of your project page:

    <script src= "https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.6.0/p5.js"></script>
    <script src= "http://cdnjs.cloudflare.com/ajax/libs/p5.js/0.5.6/addons/p5.dom.js"></script>

    Your code now speaks P5!

  2. In the setup() function, create a canvas and set your background color to something light.

  3. It's time to draw a tree! First draw a trunk using the line() or rect() functions. The base of your trunk should be at the screen's center. Make sure to color it nicely as well! Once you have your trunk, give it some leaves using the ellipse() function. Make sure to color your leaves as well.

  4. Our terrarium should have many trees though. Much like a real forest we'll have trees of different sizes, ages, and shapes. To make this possible we will use a Javascript class. Your tree class should start out simply with something like:

    // This is the syntax to declare a new class class Tree { // This function will draw our tree for us show() { // Your trunk and leaves code can go here } }


  5. With our Tree class in hand, we can create a new tree easily using the new keyword. Try creating your tree and calling its show() function.

  6. Now let's give each tree some additional properties. We can do that by using our class's constructor(). Tree's constructor is called everytime we call new and can set initial properties for that new tree. Use it to assign input x and y positions, generate a random leafColor, and set a default initial size.

    // This is the syntax to declare a new class class Tree { // constructor is called whenever new is used to create a tree constructor(x, y) { // Set the tree's x and y positions to properties here // Generate a leafColor property // Set a constant initial size } // This function will draw our tree for us show() { // Your trunk and leaves code can go here // Use the tree's x and y position properties to place the tree // Use the tree's leafColor to color the leaves // Use the tree's size to scale the tree } }

  7. In your setup() function, create 50 trees at random positions to generate your forest!

  8. Let's begin growing our trees. Give the Tree class a size property and use it to control the size of its trunk and leaves when you draw it.

  9. Create an age() function that is called in your draw() function. This function should increase your tree's size, causing your tree to grow overtime

  10. At this point, you'll have created a forest of immortal trees that grow indefinitely. To prevent that, let's stop our trees from living forever. Create a currentAge property for our Tree class. In our age() function increment the currentAge property. Finally, set a maxAge for trees. Trees with a currentAge beyond their maxAge should no longer call show(). BONUS: How can we fade our trees gently away, rather than having them suddenly disappear?

  11. Let's add some more colors to our forest. Change a Tree's leaf color based on its currentAge to show time is passing. BONUS: How can we make the color changes smooth?

  12. We should now have a vibrant forest full of colors and activity. However, a forest is not full of identical trees. Add some randomization to your trees so that each one is different. For example, you could vary the number of leaves, their positions, or anything you can think of!

  13. Now our forest is looking beautiful and varied, but a forest is full of multiple varieties of trees such as pines, oaks, birch, maple and more! We can use classes to help us create whole new varieties of trees. To begin let's clean up our Tree class by moving some code around. Create three new functions: showTrunk() (which shows the tree's trunk), showLeaves() (which shows the tree's leaves), and leafColor() (which pick's the tree's leaf color based on its age). Call these new functions appropiately in your show() function and ensure your forest functions the same way.

  14. Create a new subclass of our tree function. A subclass is a new class that inherits properties and functions from its superclass. If you want to change the behaviour of a superclass you can overwrite it's functions in your subclass. Your tree class should start out simply with something like:

    // Our new subclass TreeVariety, inherits functions and properties from its superclass Tree class TreeVariety extends Tree { // This will overwrite Tree's leafColor(), omit if you want it the same leafColor() { // Add code here if you want colors to behave differently. perhaps an Evergreen? } // This will overwrite Tree's showTrunk(), omit if you want it the same showTrunk() { // Add code here to change the the Tree's trunk, perhaps a birch tree? } // This will overwrite Tree's showLeaves(), omit if you want it the same showLeaves() { // Add code here to change the the Tree's trunk, perhaps a maple tree? } }

  15. When we spawn a tree, randomly choose which type of tree you want to spawn and add to the trees array. Now our forest is colorful, vibrant, and full of different trees. Experiment, add trees or variations, or whatever you like until your forest is perfect!
